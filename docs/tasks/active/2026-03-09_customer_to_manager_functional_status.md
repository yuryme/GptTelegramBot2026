## Current Product Goal

Telegram-бот должен понимать обычный русский язык, создавать напоминания по запросу пользователя и присылать их в нужное время в Telegram.

## Implemented

- [x] Текстовые команды на русском языке.
- [x] Голосовые сообщения через STT с дальнейшей обработкой как обычного текста.
- [x] Schema-first обработка LLM-команд с Pydantic-валидацией.
- [x] Создание одиночных, множественных и циклических напоминаний.
- [x] Просмотр напоминаний: все, на сегодня, по фильтрам.
- [x] Удаление по фильтру, по `#ID`, по последним `N`.
- [x] Soft-delete без физического удаления.
- [x] Фоновая отправка напоминаний планировщиком.
- [x] Аудит действий пользователя.
- [x] Кнопки `/start`, быстрые действия, раздел настроек.
